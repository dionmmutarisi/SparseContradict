# SparseContradict

**SparseContradict** is a synthetic benchmark for measuring how the contradiction-detection accuracy of language models degrades as a function of two difficulty knobs: (1) the *distance* in sentences between two contradicting statements inside a financial document, and (2) the *number of distractor pairs* (near-contradictions that are actually consistent) seeded into the same document. All documents are generated by a large language model using a fictional financial domain, and all models are evaluated zero-shot with no training or fine-tuning.

---

## Research Question

> How does the contradiction-localisation accuracy of language models degrade as a function of (a) the sentence-level distance between the contradicting statements and (b) the density of distractor pairs in the document?

---

## Benchmark Design

| Parameter | Values |
|---|---|
| Document length | 52–65 sentences (sampled uniformly per document) |
| Distance levels | 5, 10, 20, 30, 40, 48 sentences |
| Distractor count levels | 0, 2, 4, 6, 8 pairs |
| Documents per cell | 25 |
| Total documents | 750 (6 × 5 × 25) |
| Domain | Synthetic fictional financial companies |
| Task | Extractive contradiction localisation (zero-shot) |

### Models under test

| Model | Type |
|---|---|
| GPT-4o-mini | API (OpenAI) |
| Llama 3.1 8B Instruct | Local (HuggingFace, 4-bit) |
| Mistral 7B Instruct v0.3 | Local (HuggingFace, 4-bit) |
| Phi-3.5 Mini Instruct | Local (HuggingFace, 4-bit) |

## Setup

```bash
# 1. Clone the repository
git clone <repo-url>
cd SparseContradict

# 2. Create and activate a virtual environment
python3 -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure API keys
cp .env.example .env
# Edit .env and fill in your keys:
#   ANTHROPIC_API_KEY=<your-key>   (required for generation)
#   OPENAI_API_KEY=<your-key>      (required for gpt-4o-mini evaluation)
```

---

## Running

```bash
# Step 1: Generate the 1,500-document dataset
python run_generate.py

# Step 2: Evaluate a model (repeat for each model)
python run_evaluate.py --model gpt-4o-mini
python run_evaluate.py --model llama-3.1-8b
python run_evaluate.py --model mistral-7b
python run_evaluate.py --model phi-3.5-mini

# Step 3: Produce all figures
python run_plot.py
```

Each evaluation run is **resumable**: if interrupted, re-running `run_evaluate.py` with the same `--model` flag will skip already-evaluated documents.

## Output

### `data/documents.jsonl`

One JSON object per line. Each record contains:

| Field | Type | Description |
|---|---|---|
| `doc_id` | string | Unique ID: `d{distance}_k{distractors}_{index}` |
| `sentences` | list[str] | All sentences of the document |
| `contradiction_pair` | [int, int] | 1-based indices of the planted contradiction |
| `distractor_pairs` | [[int, int], …] | 1-based indices of distractor pairs |
| `distance` | int | Sentence distance of the contradiction pair |
| `distractor_count` | int | Number of distractor pairs |
| `n_sentences` | int | Total sentences in this document |

### `results/{model}.jsonl`

One JSON object per evaluated document:

| Field | Type | Description |
|---|---|---|
| `model` | string | Model name |
| `doc_id` | string | Document identifier |
| `distance` | int | Contradiction distance |
| `distractor_count` | int | Number of distractor pairs |
| `n_sentences` | int | Document length |
| `raw_response` | string | Raw model output |
| `prediction` | [int, int] or null | Parsed predicted pair |
| `correct` | bool | Whether prediction matches ground truth |

### `figures/`

| Figure | Description |
|---|---|
| `figure1_accuracy_vs_distance` | Line plot: mean accuracy vs. distance, one line per model, 95% CI bands, chance baseline |
| `figure2_heatmap_grid` | 2×2 heatmap grid: accuracy across all (distance, distractor) cells per model |
| `figure3_degradation_slopes` | Coefficient plot: OLS slope of accuracy ~ distance with 95% CI per model |

Each figure is saved as both `.pdf` (for the thesis) and `.png` (for quick inspection) at 300 DPI.
